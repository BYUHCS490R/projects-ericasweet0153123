Root is the top-level folder on a website. Folder and file names form the path, and the path shows the location from the root to the file.

The default document is the file that is automatically opened when a user only types the folder name only. Usually, index.html is the default document.

An absolute link is a link that includes the whole URL. A relative link is a link that is based on the current location.
Relative links are used because they are managed easily when we move or copy files within the same website.
