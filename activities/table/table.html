The <table> tag is used when we express data in a row-and-column structure. We should use table tag only when the data has a tabular meaning. For example, we can use it for grade table or schedule table.

On the other hand, we should not use the <table> tag for layout purposes. In the past, we used table tags to design web pages. However, nowadays we design web pages with CSS.

We should use additional attributes to make a table accessible to screen readers. Firstly, we can make a structure by using <thead>, <tbody>, <tfoot> tags. Secondly, we can indicate the purpose of the table by using the <caption> tag. Finally, if the description is located on the left side rather than at the top, we can put id attribute in the description tag and put header properties in the data cell that should be linked to the header cell. This helps screen readers convey the meaning of each data cell.
