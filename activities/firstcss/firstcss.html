CSS

The difference between inline, embedded, and external CSS is how styles are applied to a web page.

  • Inline CSS is written directly in HTML tags.
  • Embedded CSS uses the <style> tag inside the <head> section to apply in the whole HTML document.
  • External CSS involves creating a separate CSS file and linking it to the HTML file.

The reason why we need to use External CSS is these:

  • External CSS is easy to manage and clear.
  • By editing a CSS file, we can reflect on the changes in multiple HTML files at the same time.
  • Because the code is separated, HTML file will be easy to read.

Selector is a tool that helps us to choose what kind of HTML element we will apply a style. By using selector, we can designate styles based on tags, classes, and id numbers.
