Structural Tags

Structural tags are the tags that organize the structure of a webpage.

Here are the elements that make up the structure.

  • header tag
  • navigation tag
  • main tag
  • footer tag

The <header> element represents the section header. Title, logos, navigation can be included here.

The <nav> element contains site navigation menus. Generally it shows lists of links. This tag can go in the header, but can be located below the header also.

The <main> element contains the primary content the webpage.

The <footer> element shows the bottom of the page. Contact information, copyright, and additional links can be included.
