Background Images

  • Background-image is used for decoration. Generally, it is used when the content still makes sense without the image.
    Also, by using CSS, we can use it when we want to control the position, size, and repetition.
  • IMG tag is used for essential information or content. It is used when a screen reader needs to read the image, in other words, when we consider accessibility for the visually impaired.
    The image element can be detected through search engines and be included in image search results.
