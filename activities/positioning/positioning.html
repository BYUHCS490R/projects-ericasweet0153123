Positioning

Normal flow is a flow that every element in HTML are placed from top to bottom, and left to right.

Float surrounds elements by pushing them to left or right. Clearing the float is needed because it helps next elements not overlap.

Absolute designates exact position depending on the parent element. Relative moves its original position. Fixed doesn't move even though we scroll down or scroll up the webpage.

For mobile devices, we can make the buttons' size bigger, add spacing, and use menu icons to make toggle menus using CSS.
