Difference between block tags and inline tags

Block tags start with new lines, and end with new lines. They occupy the whole width of the parent tags.
We can control the size with width and height property. Also, margin for up, down, left and right is all applied.
However, Inline tags don't start with new lines. They follow the flow of text. They take up the width for only the portion they need. Also, the margin is applied for only left and right.

Margin, Border, and Padding

Padding, border, and margin are main parts of CSS Box Model. Padding reveals the blank between content and border.
Border means the outline around the padding. Margin is blank outside of the border.

Units of measurement

These are the units of measurement that I can use in CSS. px means pixels and it's the most common unit. We can use physical units like cm, mm, in. Also, % is the size of parent units.

Figure tag

Figure tags are useful when independent contents such as image, illustration, quote, etc. are used to describe or group.
