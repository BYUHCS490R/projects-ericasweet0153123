CSS Selectors

1. Here are the different ways to select an item.

  • Tag Name selector: such as p { }, we can choose the entire tag type.
  • ID selector: such as #idName { }, we can select an element in a form.
  • Class selector: like .className { }, we can apply uniquely in a document.
  • Children/descendent selector: such as div p { }, or header h1 { }, we can also pick different elements within a parent element.

2. ID can be applied to only one, but Class can be applied multiple times.

3. A span tag is an inline element. We can use it when we want to change the style of part of text.
In contrast, a div tag is a block element. It is used when we divide paragraphs or large sections to make it structurally.

4. A div tag is a generic tag that doesn't have any meaning. It is used for applying style.
However, header, main, and footer tags are semantic tags, which means they have meanings and roles.
Header tag includes titles, logos, navigation. Main tag includes main content.
Lastly, footer tag contains contact information and other details.

5. Here are some different ways to select a color.

  • (1) By name: color: blue;
  • (2) RGB value: color: rgb(213, 91, 115);
  • (3) Hex code: color: #FF0077;
  • (4) RGBA value: color: rgba(255, 0, 0, 0.5);
