  • JavaScript is a programming language that makes webpages interactive. By using JavaScript, clicking buttons, opening menus, and validating user input are possible. Basically, the company called “Netscape” originally named it as “LiveScript”, but at that time, Java was so popular so that they were led to name it as JavaScript. The name is like Java, but it's a completely different language.
  • Two methods to run JavaScript are first, in a web browser (behind DOM environment), second, in Node.js environment (outside of the browser). In a web browser, it runs with html, and it runs in chrome, edge, firefox, and safari. In Node.js environment, you can build servers, and you can read/write files.
  • The most popular front-end js library was jQuery. From 2006 to 2015, it was almost a standard for web. This was popular because it made DOM manipulation easily.
  • The most current popular front-end js library is React. It was developed by facebook, and it's most in-demand in job market. It is used as a standard for SPA (Single Page Application).
