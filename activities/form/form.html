  • There are many ways to style a form. In CSS, we can make a website look better by using the padding, margin, and background-color properties. Also, by using “display: flex”, we can arrange each row horizontally and adjust the spacing with the gap properly.
  • Flex allows the website to stay responsive and visually balanced regardless of the screen size. On desktop, by using “display: flex”, we can align elements horizontally, making the labels and input fields well organized. On mobile, with @media (e.g. max-width: 480px) and flex-direction: column, we can make the layout more readable.
