Internet

1. Internet is a huge path that connects computers all over the world.

Web

2. Web is a space of information on the internet. We can see a website because of this.

How are the Internet and Web different?

3. Internet is like a path that connects computers, but the web is a way to see information on the path.

ASCII and Unicode

4. ASCII and Unicode are ways to change letters into numbers so that computers can read. ASCII can only express the alphabets and numbers. In comparison, Unicode can express almost all languages around the world.

What is UTF-8

5. UTF-8 is one of the methods for saving Unicode. Thanks to UTF-8, languages such as English, Korean can be displayed properly.

URL

6. URL is an address that allows us to go into a website.

Domain Name

7. A domain Name is a name that shows where to go within a URL. It is made with names that people can easily understand.

HTTP

8. HTTP is a promise or rule for receiving and giving information on a website.

HTTPS

9. HTTPS is a combination of HTTP and security, so that people can exchange information safely.

IP Address

10.IP address is a kind of home address on the internet. Every computer and phone have individual IP address.

DNS

11. DNS is in charge of changing names that people can easily understand into numerical addresses.

W3C Recommendations

12. W3C Recommendations are global rules that should be kept while making a website.

XML, XHTML, HTML, HTML5

13. XML is a language for saving and conveying data. XHTML is a strict version of HTML so keeping rules is important. HTML is the basic language used to make a website. HTML5 is the latest version, and it allows us to put videos and music easily.
